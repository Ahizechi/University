-----------------------------------------------------------------------
To launch the nodes using roslaunch:
    roslaunch beginner_tutorials calculator.launch
To launch the nodes individually:
    rosrun beginner_tutorials listener.py [Node 3]
    rosrun beginner_tutorials add_two_ints_server.py [Node 2]
    rosrun beginner_tutorials add_two_ints_client.py [Node 1]
-----------------------------------------------------------------------
2 Text files will be created:
    AllAnswers.txt contains the expressions, commands, answers, 
    operation number and the errors. Each user session is seperated by
    the date and time.
    Answers.txt contains the expressions, errors and answers.
-----------------------------------------------------------------------
List of useable commands:
History: Displays the list of transactions and errors.
Clear: Clears the log.
Elements: Displays the number of elements in the log.
Recall: Recalls a specific line from the log.
Time: Displays system run time, date and time.
-----------------------------------------------------------------------
You can see what operations can be performed and how to use them in 
the 'How_To_Use' text file. The 'How_To_Use' text file also prints 
to the screen (see demo video).
